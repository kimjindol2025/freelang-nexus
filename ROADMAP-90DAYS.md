```markdown
## Supported Type Mappings

### Integer Types
| Rust | V | Go | C | Julia |
|------|---|----|-|-------|
| i32  | i32 | C.int | int | Int32 |
| i64  | i64 | C.longlong | long long | Int64 |
| u32  | u32 | C.uint | unsigned int | UInt32 |

| Rust | V | Go | C | Julia |
|------|---|----|-|-------|
| f32  | f32 | C.float | float | Float32 |
| f64  | f64 | C.double | double | Float64 |

### String/Pointer Types
| Source | Target | Conversion | Memory |
|--------|--------|-----------|--------|
| Rust: &str | C: char* | CString | Automatic |
| Go: string | C: char* | C.CString() | Manual free() |
| Python: str | C: char* | PyUnicode_AsUTF8 | Python owns |

## Unsupported Combinations
- Rust: Vec<T> → C: array (❌ Use raw pointer)
- Python: list → Go: slice (❌ Manual iteration)
- Julia: Matrix → C: array (❌ Flatten first)
```

```typescript
// src/nexus/codegen/type-validator.ts (신규)

class TypeValidator {
  validateCall(sourceType: Type, targetType: Type): ValidationResult {
    // Level 1: Exact match
    if (this.exactMatch(sourceType, targetType)) {
      return { valid: true };
    }
    
    // Level 2: Implicit conversion (e.g., i32 → i64)
    if (this.implicitConversion(sourceType, targetType)) {
      return { valid: true, warning: "implicit conversion" };
    }
    
    // Level 3: Incompatible
    return {
      valid: false,
      error: `Cannot convert ${sourceType} to ${targetType}`,
      suggestion: this.suggestFix(sourceType, targetType)
    };
  }
  
  suggestFix(sourceType: Type, targetType: Type): string {
    // "Rust Vec<i32> → C array"면
    // → "Use raw pointer: &vec[0] as *const i32"
  }
}
```

### Week 5–6: 병렬 컴파일 + 실패 격리

```typescript
// runner.ts 개선

async function buildLanguagesInParallel(langs: Language[]): Promise<BuildResult[]> {
  const independentLangs = buildDependencyGraph(langs)
    .getIndependentGroups();
  
  const results = [];
  for (const group of independentLangs) {
    // 같은 그룹의 언어들은 병렬 컴파일
    const parallelResults = await Promise.all(
      group.map(lang => compileLang(lang))
    );
    results.push(...parallelResults);
    
    // 실패 여부 확인
    const failures = parallelResults.filter(r => !r.success);
    if (failures.length > 0) {
      // 실패한 언어 로그
      for (const failure of failures) {
        console.error(`❌ ${failure.language}: ${failure.error}`);
      }
      // 다른 언어는 계속 컴파일
    }
  }
  
  return results;
}
```
