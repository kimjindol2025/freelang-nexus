### ✅ 1. Deterministic Build: 10회 반복 빌드 해시 일치

**검증 방법**:
```bash
#!/bin/bash
# verify-deterministic-build.sh

HASHES=()
for i in {1..10}; do
  npm run build > /dev/null 2>&1
  HASH=$(md5sum main | cut -d' ' -f1)
  HASHES+=("$HASH")
done

# 모든 해시가 같은지 확인
UNIQUE_COUNT=$(printf '%s\n' "${HASHES[@]}" | sort | uniq | wc -l)

if [ "$UNIQUE_COUNT" -eq 1 ]; then
  echo "✅ PASS: 10/10 빌드 해시 일치"
  exit 0
else
  echo "❌ FAIL: 해시 불일치 감지"
  printf '%s\n' "${HASHES[@]}"
  exit 1
fi
```

**기준**: 10회 모두 동일 해시  
**현재 상태**: ⚠️ 미달성  
**마감**: 2026-04-30
